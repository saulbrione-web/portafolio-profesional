Diseño Orientado a Conversión (Barbería)

Problema: El negocio perdía clientes porque solo usaban redes sociales y los mensajes se perdían.
Solución: Desarrollo de una Landing Page de carga ultra rápida (menos de 2 segundos) con diseño atractivo y un llamado a la acción directo a WhatsApp, aumentando las reservas inmediatas.

Sistema de Gestión y Base de Datos

Problema: Descontrol en el inventario y pérdida de datos de clientes por usar hojas de cálculo lentas.
Solución:Programación de un panel administrativo privado (Dashboard). Utilizo arquitectura backend con Python y almacenamiento estructurado en bases de datos relacionales para garantizar integridad, seguridad y velocidad.
